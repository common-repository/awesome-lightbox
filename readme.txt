== Installation ==
1. Log in and navigate to _Plugins → Add New.
2. Type “Awesome Lighgtbox into the Search and hit Enter.
3. Locate the Awesome Lightbox plugin in the list of search results and click Install Now.
4. Once installed, click the Activate link.

It can also be installed manually.

1. Download the Awesome Lightbox plugin from WordPress.org.
2. Unzip the package and move to your plugins directory /wp-content/plugins/awesome-lightbox.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Awesome Lightbox in the list and click the Activate link.
